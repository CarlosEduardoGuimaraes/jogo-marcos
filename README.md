# jogo-marcos

O jogo é bem simples. Um jogo de terror inspirado nos antigos silent hills e Resident evils, porém, feito no rpg maker.

Comandos:
Andar - setas
Correr - Shift
Interagir - enter
Menu e inventário - ESC

Fluxo máximo e mínimo:
Utilizado no script personalizado de corrida
Utilizado pelo seguinte motivo: Naturalmente já seria uma necessidade no jogo, jogos de terror não existem sem corrida, e um sistema que evita spam de corrida é essencial.

Travelling Salesman Problem:
Utilizado no sistema do item personalizado de busca de objetivos no jogo
